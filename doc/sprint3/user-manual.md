## สำหรับผู้ใช้ทั่วไป (User)

### 1. เข้าหน้าแจ้งเหตุการณ์
- Login เข้าสู่ระบบ
- คลิกเมนู **"แจ้งเหตุการณ์"** ที่ Navbar
- หรือเข้าผ่าน URL: `http://localhost:3001/myReports`

### 2. สร้างรายงานใหม่
#### ขั้นตอนที่ 1 — เลือกTrip ที่เกี่ยวข้อง (ไม่บังคับ)
เลือก Trip ที่คุณต้องการแจ้งรายงาน (ทั้งในฐานะผู้โดยสาร)

#### ขั้นตอนที่ 2 — เลือกระดับความสำคัญ
คลิกปุ่มระดับความสำคัญ:
| ปุ่ม | ระดับ | ใช้เมื่อ |
|------|-------|---------|
| 🟢 ต่ำ | LOW | เรื่องทั่วไป ไม่เร่งด่วน |
| 🟡 ปานกลาง | MEDIUM | **ค่าเริ่มต้น** — เรื่องที่ควรแก้ไข |
| 🟠 สูง | HIGH | เรื่องเร่งด่วน ต้องดำเนินการเร็ว |
| 🔴 วิกฤต | CRITICAL | เรื่องวิกฤต ต้องแก้ไขทันที |

#### ขั้นตอนที่ 3 — เลือกประเภทเหตุการณ์ (จำเป็น)
เลือกจาก dropdown:
| ตัวเลือก | สำหรับ |
|----------|--------|
| ความปลอดภัย | สำหรับแจ้งเหตุที่กระทบต่อสวัสดิภาพ เช่น ขับรถอันตราย หรือคุกคาม |
| ปัญหาระหว่างเดินทาง | สำหรับแจ้งปัญหาทั่วไป เช่น รถเสีย, น้ำมันหมด, หรือหลงทาง |
| พฤติกรรม | สำหรับแจ้งเรื่องมารยาท หรือพฤติกรรมที่ไม่เหมาะสมของคู่กรณี |
| ทรัพย์สิน | สำหรับแจ้งเรื่องลืมของไว้บนรถ หรือทรัพย์สินเสียหาย |
| เทคนิค | สำหรับแจ้งปัญหาเกี่ยวกับตัวแอปพลิเคชัน หรือระบบการใช้งาน |
| อื่นๆ | ปัญหาอื่นๆ |


#### ขั้นตอนที่ 4 — กรอกหัวข้อ (จำเป็น)
สรุปปัญหาสั้นๆ

#### ขั้นตอนที่ 5 — กรอกรายละเอียด (จำเป็น)
อธิบายเหตุการณ์ที่พบอย่างละเอียด ไม่เกิน 2000 ตัวอักษร

#### ขั้นตอนที่ 6 — แนบรูปภาพ (ไม่บังคับ)
- คลิก **เลือกรูป** เพื่อเลือกไฟล์ภาพ
- รองรับ JPG, PNG ขนาดไม่เกิน 5MB
- สามารถดูพรีวิวรูปก่อนส่ง
- กด ✕ เพื่อลบรูปที่เลือก

#### ขั้นตอนที่ 7 — ปักหมุดตำแหน่ง (จำเป็น)
- **คลิกบนแผนที่** ที่ตำแหน่งที่เกิดเหตุ
- จุดหมุดสีแดงจะปรากฏ
- สามารถ**คลิกใหม่**เพื่อย้ายหมุด
- กด **✕ ลบตำแหน่ง** เพื่อเริ่มใหม่
- พิกัด (lat, lng) จะแสดงด้านล่างแผนที่


#### ขั้นตอนที่ 8 — ส่งรายงาน
- กดปุ่ม **ส่งรายงาน**
- ถ้าสำเร็จ จะแสดงข้อความ ✅ "ส่งรายงานเรียบร้อย!"
- ถ้าข้อมูลไม่ครบ จะแสดงข้อความ error บอกว่าขาดอะไร

### 3. ดูรายงานของตัวเอง
- เลื่อนลงในหน้าเดียวกัน จะเห็นรายการรายงานทั้งหมดที่เคยแจ้ง
- แต่ละรายงานแสดง:
  - 🏷️ ประเภทเหตุการณ์ (badge สี)
  - ⚡ ระดับความสำคัญ
  - 📊 สถานะ (รอ/กำลังดำเนินการ/แก้ไขแล้ว/ปฏิเสธ)
  - 📝 หมายเหตุจาก Admin (ถ้ามี)
  - 📍 ตำแหน่ง (ถ้าปักหมุด)
  - 🖼️ รูปภาพ (คลิกเพื่อขยาย)


### 4. การแจ้งเตือน (Notifications)
- เมื่อ Admin มีการอัปเดตสถานะรายงาน (เช่น รับเรื่อง, แก้ไขแล้ว)
- ผู้ใช้จะได้รับการแจ้งเตือนผ่าน **กระดิ่งแจ้งเตือน** (Bell Icon) มุมขวาบน
- สามารถคลิกที่การแจ้งเตือนเพื่อดูรายละเอียดรายงานได้ทันที
- หาก Admin ปฏิเสธรายงาน จะมีหมายเหตุแจ้งเหตุผลให้ทราบ

### 5. แชทกับ Admin
- ในหน้ารายการรายงาน กดปุ่ม **แชท** ที่รายงานที่ต้องการ
- หน้าต่างแชทจะเปิดขึ้นมา
- พิมพ์ข้อความที่ต้องการสอบถามเพิ่มเติม แล้วกด Enter หรือปุ่มส่ง
- Admin จะตอบกลับในช่องทางนี้
- **หมายเหตุ:** แชทจะใช้งานได้เมื่อสถานะเป็น PENDING หรือ IN_PROGRESS

## สำหรับผู้ดูแลระบบ (Admin)

### 1. เข้าหน้าจัดการรายงาน
- Login เข้าสู่ระบบด้วยบัญชี Admin
- คลิก **Report Management** ใน Sidebar
- หรือคลิก **"ดูรายงานทั้งหมด"** ที่ Navbar
- หรือเข้าผ่าน URL: `http://localhost:3001/admin/reports`

### 2. กรองรายงาน
ใช้ filter ด้านบน:
- **สถานะ:** ทั้งหมด / PENDING / IN_PROGRESS / RESOLVED / REJECTED
- **ประเภทเหตุการณ์:** ทั้งหมด / DRIVER / PASSENGER / ROUTE / BOOKING / SYSTEM / OTHER
- **ระดับความสำคัญ:** ทั้งหมด / LOW / MEDIUM / HIGH / CRITICAL
- **เรียงลำดับ:** ค่าเริ่มต้น / ความสำคัญ
- **ค้นหา:** พิมพ์คำค้นหาแล้วกด **ค้นหา**
- กด **ใช้ตัวกรอง** เพื่อกรอง / **ล้างตัวกรอง** เพื่อเคลียร์


### 3. ดูรายงานทั้งหมด
ตารางแสดงรายงานทุกรายการ:
| คอลัมน์ | รายละเอียด |
|---------|-----------|
| ผู้แจ้ง | ชื่อ + อีเมล |
| ประเภท | badge สีตามประเภทเหตุการณ์ |
| ความสำคัญ | badge สีตามระดับ |
| หัวข้อ | หัวข้อรายงาน |
| สถานะ | badge สีตามสถานะ |
| วันที่แจ้ง | วันเวลาที่แจ้ง |
| การกระทำ | ปุ่ม ดู / ลบ |

### 4. ดูรายละเอียดรายงาน
- คลิก **ดู** ที่รายงานที่ต้องการ
- หน้ารายละเอียดแสดง:
  - ข้อมูลผู้แจ้ง (ชื่อ, อีเมล, เบอร์โทร)
  - ประเภท + ความสำคัญ + สถานะ (ทั้ง 3 badge)
  - หัวข้อ + รายละเอียด
  - 🖼️ รูปภาพ (คลิกเพื่อขยายใน Lightbox)
  - 🗺️ แผนที่แสดงตำแหน่งเหตุการณ์

### 5. อัปเดตสถานะ + หมายเหตุ
- ส่วนล่างของหน้ารายละเอียด
- เลือก **สถานะ** จาก dropdown:
  - `PENDING` → รอดำเนินการ
  - `IN_PROGRESS` → กำลังดำเนินการ
  - `RESOLVED` → แก้ไขแล้ว
  - `REJECTED` → ปฏิเสธ
- กรอก **หมายเหตุ** (ผู้แจ้งจะเห็นข้อความนี้)
- กด **บันทึก**


### 6. ลบรายงาน
- คลิก **ลบ** ที่รายงานในตาราง
- ยืนยันการลบ
- รายงานจะถูกลบออกจากระบบถาวร

### 7. Profile Dropdown (Admin Header)
- Hover ที่ชื่อ profile มุมขวาบน
- เมนูจะแสดง:
  - **บัญชีของฉัน** → ไปหน้า profile
  - **Dashboard** → กลับหน้า admin
  - **Logout** → ออกจากระบบ

### 8. แชทคุยกับผู้แจ้ง
- ในหน้ารายละเอียดรายงาน เลื่อนลงมาด้านล่างสุด
- จะพบส่วน **แชทกับผู้แจ้ง** 
- สามารถพิมพ์ข้อความเพื่อสอบถามรายละเอียดเพิ่มเติมจากผู้แจ้งได้
- ข้อความจะส่งไปถึงผู้แจ้งแบบ Real-time
- ประวัติการแชทจะถูกบันทึกไว้ตลอด

### 💳 ข้อมูลการชำระเงิน (Payment Information)
🔹 หน้ารายการเอกสารการชำระเงิน
คำอธิบาย:
หน้าจอนี้แสดง ข้อมูลการชำระเงินของการจอง (Payment Information) โดยประกอบด้วย:
📍 เส้นทางการเดินทาง (ต้นทาง → ปลายทาง)
📅 วันที่เดินทาง
🎟️ จำนวนที่นั่ง
💰 ราคาที่ชำระ
📄 รายการเอกสารการชำระเงิน

ผู้ใช้สามารถดูเอกสารที่เกี่ยวข้องกับการชำระเงิน ได้แก่:
ใบสำคัญรับเงิน (Receipt Voucher)
ใบกำกับภาษีแบบย่อ (Tax Invoice)

⚙️ การทำงานของปุ่ม
👁️ ดูเอกสาร → เปิดดูรายละเอียดเอกสาร
⬇️ ดาวน์โหลด → ดาวน์โหลดไฟล์เอกสาร
🔄 สร้างเอกสารใหม่ → สร้างเอกสารการชำระเงินใหม่
❌ ปิดหน้าต่าง → ออกจากหน้านี้

🔹 หน้ารายละเอียดการจอง (แสดงข้อมูลการชำระเงิน)
คำอธิบาย:
หน้านี้แสดง รายละเอียดการชำระเงินของ Trip ได้แก่:
💰 ราคาทั้งหมด (เช่น 100 บาท)

📊 สถานะการจอง (เช่น “ยืนยันแล้ว”)

### 💳 ข้อมูลการรับเงิน (Receiving Payment Information)
คำอธิบาย:
หน้านี้ใช้สำหรับตั้งค่า ช่องทางรับเงิน ของผู้ใช้งาน (เช่น คนขับ) เพื่อให้ผู้โดยสารสามารถโอนเงินหรือชำระเงินได้อย่างถูกต้อง

🔹 1. PromptPay
ใช้สำหรับรับเงินผ่าน PromptPay
วิธีตั้งค่า:
กรอกข้อมูลอย่างใดอย่างหนึ่ง:
📱 เบอร์โทรศัพท์
🆔 เลขบัตรประชาชน
🧾 เลขประจำตัวผู้เสียภาษี
กดปุ่ม บันทึก
📌 ระบบจะแสดง:
👤 ชื่อผู้รับเงิน (ตรวจสอบอัตโนมัติ)

🔹 2. QR Code (ไฟล์)
ใช้สำหรับให้ผู้โดยสารสแกนจ่ายเงิน
วิธีอัปโหลด:
กดปุ่ม เลือกไฟล์
รองรับไฟล์:
JPG, PNG, PDF
ขนาดไม่เกิน 10 MB
เมื่อเลือกไฟล์:
ระบบจะแสดงตัวอย่าง (Preview)
กดปุ่ม อัปโหลด QR
📌 เหมาะสำหรับ:
การจ่ายเงินที่รวดเร็ว
ลดความผิดพลาดจากการกรอกเลขบัญชี

🔹 3. บัญชีธนาคาร
ใช้สำหรับรับเงินผ่านการโอนเข้าบัญชีธนาคาร
วิธีเพิ่มบัญชี:
กดปุ่ม + เพิ่มบัญชี
เลือก:
🏦 ธนาคาร
กรอก:
🔢 เลขบัญชี (ตัวเลขเท่านั้น)
ระบบจะแสดง:
👤 ชื่อบัญชี (อัตโนมัติ)
กดปุ่ม เพิ่มบัญชี

🔹 4. รายการบัญชีธนาคาร
หลังจากเพิ่มบัญชีแล้ว ระบบจะแสดงรายการ:
🏦 ชื่อธนาคาร
🔢 เลขบัญชี
👤 ชื่อเจ้าของบัญชี
การจัดการ:
🗑️ ลบบัญชี
📄 ดูรายละเอียดบัญชี
