# Sprint 2 - PaiNamNae Web App (Group 2 Sec 3)

## ฟีเจอร์ที่ทำสำเร็จใน Sprint 2

### 1. ระบบแชทอัจฉริยะในหน้ารายงานเหตุ (Enhanced Chat System)

- **รองรับการส่งรูปภาพและไฟล์:** ผู้ใช้งานและผู้ดูแลระบบสามารถส่งรูปภาพหลักฐานหรือไฟล์เอกสารเพิ่มเติมผ่านช่องแชทได้แล้ว
- **ระบบพรีวิวไฟล์:** เพิ่มส่วนแสดงผลภาพตัวอย่าง (File Preview) ก่อนกดส่ง เพื่อตรวจสอบความถูกต้อง
- **Cloudinary Integration:** ระบบอัปโหลดไฟล์มีความเสถียรและปลอดภัย โดยจัดเก็บไฟล์ไว้บน Cloud
- **Real-time Communication:** พัฒนาการส่งข้อความผ่าน Socket.IO ให้รองรับข้อมูลไฟล์แนบ ทำให้การสนทนาระหว่าง User และ Admin ลื่นไหลไม่สะดุด

### 2. ระบบแจ้งเหตุ (Reporting System)

- **Auto-detect Location:** เพิ่มความสะดวกด้วยระบบตรวจจับพิกัดตำแหน่งปัจจุบันอัตโนมัติเมื่อทำการปักหมุดในแผนที่ (Geo-location API)
- **UI Refinement:** ปรับปรุงส่วนการเลือกตำแหน่ง (Map Picker) ให้ใช้งานง่ายขึ้น และแสดงที่อยู่ (Address) จากพิกัดได้แม่นยำขึ้น

---

## รายการแก้ไขและปรับปรุง (Fixes and Improvements)

- **Schema Update:** ปรับปรุงฐานข้อมูล Prisma ให้รองรับโครงสร้างข้อความแชทรูปแบบใหม่
- **Validation:** เพิ่มการตรวจสอบขนาดไฟล์ (Max 10MB) และประเภทไฟล์ (Images) ในฝั่ง Backend เพื่อป้องกันข้อผิดพลาด
- **Input Handling:** แก้ไข Logic การส่งข้อความให้สามารถส่งได้ทั้งแบบ "ข้อความอย่างเดียว", "รูปภาพอย่างเดียว" หรือ "ส่งพร้อมกันทั้งคู่"
- **Bug Fix (Map):** แก้ไขปัญหาหมุดแผนที่คลาดเคลื่อนในบางกรณี และเพิ่มความเสถียรในการดึงข้อมูลหลังจากลากหมุด (Drag-end geocoding)

## รายละเอียดการเปลี่ยนแปลงทางเทคนิค (Technical Changes)

### Backend

- เพิ่มฟิลด์ `fileUrl`, `fileType`, และ `fileName` ใน Model `ChatMessage` (Prisma) เพื่อรองรับการเก็บข้อมูลไฟล์แนบ
- สร้าง Middleware สำหรับจัดการการอัปโหลดไฟล์ (Multer) และ Controller สำหรับอัปโหลดรูปภาพขึ้น Cloudinary โดยจำกัดขนาดไฟล์ที่ 10MB
- ปรับปรุง Socket.IO ให้รองรับการส่งข้อมูลไฟล์ผ่าน Socket เพื่อให้ข้อความแสดงผลแบบ Real-time ทั้งภาพและข้อความ

### Frontend

- เพิ่มปุ่มแนบไฟล์ (Clip icon), ส่วนแสดงผล Preview ก่อนส่ง, และการแสดงภาพ/ไฟล์ในกล่องแชท (รองรับทั้งฝั่ง User และ Admin)
- ปรับปรุงหน้า `myReports` ให้ดึงตำแหน่งปัจจุบันของผู้แจ้งโดยอัตโนมัติทันทีที่เปิดแผนที่ (Automatic Current Location Detection)
- ปรับปรุงข้อความแนะนำการใช้งานแผนที่ และเพิ่มปุ่มลบตำแหน่ง (Clear Location) ให้ชัดเจนขึ้น
