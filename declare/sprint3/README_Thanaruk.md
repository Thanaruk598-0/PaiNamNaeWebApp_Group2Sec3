**Project:** Pai Nam Nae - A Safe Ride Sharing App  
**Course:** CP353004 Software Engineering

รายงานฉบับนี้ ได้นำปัญญาประดิษฐ์ ได้แก่ **ChatGPT (GPT-5.3-Codex)** มาใช้เป็นเครื่องมือสนับสนุนในการพัฒนาและปรับปรุงระบบ ในขั้นตอนดังต่อไปนี้

1. ใช้เพื่อออกแบบและปรับปรุงตรรกะการจอง/ยกเลิกฝั่ง Backend เช่น การจำกัดสิทธิ์การยกเลิก และการคืนที่นั่งหลังยกเลิก
2. ใช้เพื่อช่วยกำหนดและตรวจสอบ Validation ของข้อมูลสำคัญ เช่น เหตุผลการยกเลิก, PromptPay (ไม่เกิน 13 หลัก) และเลขบัญชีธนาคาร (10-12 หลัก)
3. ใช้เพื่อช่วยวิเคราะห์และแก้ไขปัญหาระหว่างพัฒนา (debugging) ทั้งฝั่ง Frontend และ Backend รวมถึงการปรับ UI/UX ให้การใช้งานชัดเจนขึ้น

โดย ข้าพเจ้าได้ทำการตรวจสอบความถูกต้องของข้อมูล รวมถึงปรับปรุงและแก้ไขข้อผิดพลาดอันอาจเกิดจากผลลัพธ์ของปัญญาประดิษฐ์เรียบร้อยแล้ว
