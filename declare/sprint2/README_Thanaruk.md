**Project:** Pai Nam Nae - A Safe Ride Sharing App  
**Course:** CP353004 Software Engineering

รายงานฉบับนี้ ได้นำปัญญาประดิษฐ์ ได้แก่ ChatGPT (GPT-5.2) และ Claude (claude-sonnet-4.6) มาใช้เป็นเครื่องมือสนับสนุนในการพัฒนาและปรับปรุงระบบ ในขั้นตอนดังต่อไปนี้

1. ใช้เพื่อให้คำแนะนำในการออกแบบและปรับปรุงโครงสร้างฐานข้อมูลและส่วน Backend เช่น การกำหนด Model ด้วย Prisma การเขียน validation ด้วย Zod และการจัดการข้อผิดพลาด (error handling)
2. ใช้เพื่อช่วยวิเคราะห์และแก้ไขปัญหาที่เกิดขึ้นระหว่างการพัฒนา (debugging) เช่น ปัญหาการอัปโหลดไฟล์ การแสดงผลวิดีโอ และการเชื่อมต่อข้อมูลระหว่าง Frontend และ Backend
3. ใช้เพื่อเสนอแนวทางพัฒนาและปรับปรุงประสบการณ์ผู้ใช้ (UX) ในส่วนการอัปโหลดไฟล์หลายไฟล์ การแสดงตัวอย่างรูปภาพและวิดีโอ (preview และ lightbox) รวมถึงการกำหนด attribute ของ `<video>`
4. ใช้เพื่อช่วยออกแบบและปรับปรุงส่วนติดต่อผู้ใช้ (UI) ที่เกี่ยวข้องกับการแสดงผลสื่อประกอบ ได้แก่ หน้า Admin Detail ส่วนแนบรูปภาพ/วิดีโอ (Upload Zone) และส่วนแสดงรายการรายงานพร้อมไฟล์แนบ (Attachment Display) ในฝั่งผู้ใช้

โดย ข้าพเจ้าได้ทำการตรวจสอบความถูกต้องของข้อมูล รวมถึงปรับปรุงและแก้ไขข้อผิดพลาดอันอาจเกิดจากผลลัพธ์ของปัญญาประดิษฐ์เรียบร้อยแล้ว
