##  รายละเอียดการใช้ AI ในโปรเจกต์
ข้าพเจ้ามีการใช้งาน AI (Gemini) เพื่อสนับสนุนการทำงานในส่วนต่าง ๆ ดังนี้:

### 1. การช่วยเหลือด้านการเขียนโค้ด (Coding Assistance)
* **ส่วนที่ใช้:** ไฟล์ทดสอบ `report.api.v2.test.js`
* **ลักษณะการใช้งาน:** ใช้ในการตรวจสอบโครงสร้างคำสั่ง Jest, Supertest และการจัดการ Authentication Token
* **การตรวจสอบ:** ข้าพเจ้าได้ทำการตรวจสอบและปรับแก้โค้ดให้ทำงานได้จริงตาม Business Logic ของระบบ และผ่านการทดสอบ 100% (26 Passed)

### 2. การออกแบบการทดสอบ (Test Design)
* **ส่วนที่ใช้:** การจัดทำตาราง Test Case และ Test Data
* **ลักษณะการใช้งาน:** ใช้ AI ช่วยในการสรุป Expected Result และการระบุ IncidentType ต่าง ๆ ให้ครอบคลุม

### 3. การแก้ไขปัญหาเชิงเทคนิค (Troubleshooting)
* **ส่วนที่ใช้:** การจัดการโครงสร้าง Git และ GitHub Branch
* **ลักษณะการใช้งาน:** ใช้ปรึกษาการแก้ไขปัญหา "Embedded Repository" และการจัดระเบียบโครงสร้างโฟลเดอร์ให้ถูกต้อง

##  การรับรองความถูกต้อง
ข้าพเจ้ายืนยันว่า AI เป็นเพียงเครื่องมือสนับสนุนการทำงานเท่านั้น โดยข้าพเจ้าเป็นผู้ตรวจสอบ ปรับแต่ง และรับผิดชอบต่อผลลัพธ์สุดท้ายของงานทั้งหมดใน Sprint นี้แต่เพียงผู้เดียว
