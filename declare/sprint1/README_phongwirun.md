## 📝 Declare — การใช้ปัญญาประดิษฐ์

รายงานฉบับนี้ นำปัญญาประดิษฐ์ **Claude** มาใช้ในขั้นตอนดังต่อไปนี้

1. **ช่วยในการเขียนโค้ดบางส่วนที่เป็นภาษา JavaScript (JS)** — ช่วยแนะนำโครงสร้างโค้ด ช่วยแก้ไข Syntax 
2. **ช่วยวิเคราะห์และแก้ไข Error ที่เกิดขึ้นระหว่างการทดสอบระบบ** — Error จาก Selenium / Robot Framework, JavaScript runtime errors, DOM selector และ XPath issues
3. **ช่วยปรับปรุงโครงสร้าง** — แนะนำวิธีทำให้ test เสถียรขึ้น, วิเคราะห์สาเหตุของ Test Failure

โดยข้าพเจ้าได้ตรวจสอบความถูกต้องรียบร้อยแล้ว
